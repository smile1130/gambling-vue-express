वास्तविक नाम प्रमाणीकरण

चरण 1: कृपया सामने वाले पहचान पत्र की फोटो अपलोड करने के लिए "छवि अपलोड करें" चुनें
चरण 2: कृपया अपना वास्तविक नाम दर्ज करें
चरण 3: कृपया अपना पहचान पत्र नंबर दर्ज करें
चरण 4: "अगला" पर क्लिक करें
चरण 5：कृपया अपनी सेल्फी फोटो अपलोड करें
चरण 6: अपना सत्यापन सबमिट करने के लिए क्लिक करें

टिप्पणी:
1. कृपया तस्वीरों के लिए सरकार द्वारा जारी मूल दस्तावेजों का उपयोग करें;
2. कृपया दस्तावेज़ों को मोनोक्रोम पृष्ठभूमि पर रखें;
3. दस्तावेज़ों की तस्वीरें लेते समय, कृपया सुनिश्चित करें कि तस्वीरें स्पष्ट और अस्पष्टता या प्रतिबिंब से मुक्त हों।
4. कृपया श्वेत-श्याम फ़ोटो का उपयोग न करें।
