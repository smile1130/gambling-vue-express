BIGMUMBAI Forgot My Login Account


Dear, if you forget the login number you filled in when registering, please go to the self-service center bigmumbaics.com to submit a login account retrieval request.

1.You need to use this link https://www.bigmumbaics.com
2. After you enter the link then close the notification
3. Select issue and choose Change ID Login Password
4. Fill your BIGMUMBAI ID account
5. Fill register phone number
6. Attach photo selfie holding passbook bank
7. Attach photo selfie holding identity card
8. Fill new password (min 8 character with big-small letter and number)
9. Submit Issue



After submission, the specialist will check the information, you can check the reply status by entering the mobile phone number you submitted.
